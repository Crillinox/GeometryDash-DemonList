## 30-Second Deployment

### GitHub Pages
```
1. github.com → Sign up/Login
2. New repository → Name: gddl-website → Public → Create
3. Upload files → Drag all files → Commit
4. Settings → Pages → Source: main → Save
5. Wait 2 min → Your link: https://USERNAME.github.io/gddl-website/
```

### Netlify
```
1. netlify.com → Sign up
2. Drag entire gddl-website folder to dashboard
3. Link appears: https://your-site.netlify.app
```

### Vercel
```
1. vercel.com → Sign up
2. Add New → Project → Browse → Select gddl-website folder
3. Deploy → Link appears: https://your-project.vercel.app
```
